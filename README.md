# Fitness-Tracker
The Fitness Tracker Web App is a browser-based application that allows users to log daily fitness metrics such as steps taken, calories burned, water intake, and active minutes via an intuitive HTML form.
The application updates a dynamic history table and visualizes user progress using Chart.js, displaying the data on a responsive multi-line chart.
CSS is used to provide a clean, responsive UI, while JavaScript handles DOM manipulation, event handling, and chart rendering to ensure a seamless user experience.
Tech Stack: HTML, CSS, JavaScript, Chart.js
